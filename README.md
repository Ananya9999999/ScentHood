🌸 SCENTHOOD – AI‑Powered Perfume Recommendation Web App
SCENTHOOD is a full‑stack web application that provides personalized perfume recommendations using user preferences, mood, time, and environment.
It combines Flask, MySQL, and rule‑based AI logic to deliver explainable scent suggestions.

🚀 Features
🔐 User authentication (Register / Login / Logout)
💼 Personal perfume collection management
🧠 AI‑based perfume recommendations
📝 Clear AI explanations + confidence scores
📊 Recommendation history tracking
🎨 Elegant, responsive UI (HTML + CSS)
🗄️ MySQL database integration
