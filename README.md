<h1>🌸 ScentHood</h1>

ScentHood is a modern web application designed to help users discover, explore, and purchase fragrances through an intuitive and elegant interface. The platform focuses on creating a smooth digital experience for perfume lovers by combining clean UI, product discovery, and fragrance exploration in one place.

<h2>🚀 Features</h2>

- ✨ Fragrance Discovery

Browse a wide collection of perfumes and fragrance products.

- 🔎 Search & Filtering

Find scents easily using search functionality and category filters.

- 🛒 Product Browsing

View detailed perfume information including notes, descriptions, and images.

- 💻 Modern User Interface

Responsive design ensuring smooth experience across devices.

- ⚡ Fast and Lightweight

Optimized frontend and efficient backend structure for quick loading.

<h2>⚙️ Installation</h2>

Clone the repository:

```
git clone https://github.com/Ananya9999999/ScentHood.git
```

Move into the project folder:

```
cd ScentHood
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm start
```
